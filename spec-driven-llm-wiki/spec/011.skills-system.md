# Spec 011: Skills System

## Task/Prompt

Add a Skills system to self-improve-cli that allows instruction modules (SKILL.md files) to be discovered, loaded, and injected into agent context. Skills optionally provide OpenAI function-calling tool schemas and async handler functions. Discovery scans 6 directories for full opencode compatibility. Agent loop merges skill tools at runtime. Chat exposes `/skills` commands for listing, enabling, disabling, and installing skills.

## Tujuan

Menambahkan sistem Skills yang modular dan aman ke agent loop:

- Discovery otomatis dari 6 direktori (3 global, 3 project-level).
- Parsing SKILL.md dengan YAML frontmatter (`name`, `description`) + body instruksi.
- Skill tools menggunakan prefix `skill__<name>__<tool>` untuk menghindari konflik.
- Integrasi ke agent loop mengikuti pola yang sudah ada di `mmx-tools.js`.
- Chat commands `/skills list|enable|disable|install` untuk kontrol interaktif.
- Keamanan: handlers.js di-gate oleh `tool_policy`, default deny untuk skill tools.

## Kenapa Tujuan Ini

Agent perlu kemampuan yang bisa diperluas tanpa mengubah core runtime. Saat ini hanya `mmx-tools.js` yang bisa menambah tools secara hardcoded. Skills memungkinkan user dan project menambah instruksi dan tools secara deklaratif via file system, kompatibel dengan format opencode. Ini membuat self-improve-cli bisa di-extend oleh user tanpa forking.

## Logic Changes

1. **Discovery**: Scan 6 directories in order (global first, then project). Each directory may contain `<name>/SKILL.md` subdirectories. First match wins per skill name.

2. **Parsing**: SKILL.md has YAML frontmatter delimited by `---`. Extract `name` (string) and `description` (string). Body after frontmatter is the instruction text injected into agent context.

3. **Name validation**: Regex `^[a-z0-9]+(-[a-z0-9]+)*$`, length 1-64.

4. **Loading**: For each discovered skill, optionally load `tools.json` (array of OpenAI function-calling tool schemas) and `handlers.js` (CommonJS module exporting async functions). Tool names in `tools.json` are prefixed: `skill__<name>__<original_tool_name>`.

5. **Agent loop integration**: `runAgentTask` calls `getSkillTools(activeSkillNames)` to collect tool schemas and handlers from all active skills. These are merged into `options.tools` and `options.toolHandlers` alongside `TOOL_SCHEMAS` and `MMX_TOOL_SCHEMAS`.

6. **System prompt injection**: `systemPrompt` appends an `available_skills` block listing all discovered skills with names and descriptions. Active skills get their full instruction body injected.

7. **Profile tracking**: Active skill names stored in `overlay.profile.json` at path `/memory/active_skills` (array of strings).

8. **Tool policy**: Each skill-provided tool gets individual `tool_policy` entry using the prefixed name `skill__<name>__<tool>`. Default is `deny`. User must explicitly enable via profile or `/skills enable`.

9. **Chat commands**: New `/skills` slash command with subcommands: `list`, `enable <name>`, `disable <name>`, `install <path|url>`.

10. **Installation**: `/skills install` copies a skill directory to project-level `.opencode/skills/<name>/`. URL installation requires user confirmation.

## Pseudocode

```text
--- src/skills.js ---

SKILL_NAME_RE = /^[a-z0-9]+(-[a-z0-9]+)*$/
GLOBAL_DIRS = [
  ~/.config/opencode/skills,
  ~/.claude/skills,
  ~/.agents/skills
]
PROJECT_DIRS = [
  .opencode/skills,
  .claude/skills,
  .agents/skills
]

function discoverSkills(projectRoot):
  seen = new Set()
  results = []
  allDirs = [...GLOBAL_DIRS.map(d => resolve(homedir(), d)),
             ...PROJECT_DIRS.map(d => resolve(projectRoot, d))]
  for dir in allDirs:
    if not exists(dir): continue
    entries = readdir(dir)
    for entry in entries:
      skillFile = join(dir, entry, 'SKILL.md')
      if not exists(skillFile): continue
      if seen.has(entry): continue  // first match wins
      parsed = parseSkillFrontmatter(readFile(skillFile))
      if not SKILL_NAME_RE.test(parsed.name): skip
      if parsed.name.length > 64: skip
      seen.add(entry)
      results.push({ name: parsed.name, description: parsed.description, dir: join(dir, entry) })
  return results

function parseSkillFrontmatter(content):
  if not content.startsWith('---'): throw
  endIdx = content.indexOf('---', 3)
  if endIdx === -1: throw
  yaml = content.slice(3, endIdx).trim()
  body = content.slice(endIdx + 3).trim()
  frontmatter = parseSimpleYaml(yaml)  // key: value pairs only
  return { name: frontmatter.name, description: frontmatter.description, body }

function loadSkill(skillDir):
  skillMd = readFile(join(skillDir, 'SKILL.md'))
  parsed = parseSkillFrontmatter(skillMd)
  result = { name: parsed.name, description: parsed.description, instructions: parsed.body }
  toolsPath = join(skillDir, 'tools.json')
  if exists(toolsPath):
    rawTools = JSON.parse(readFile(toolsPath))
    result.tools = rawTools.map(t => prefixToolSchema(parsed.name, t))
  else:
    result.tools = []
  handlersPath = join(skillDir, 'handlers.js')
  if exists(handlersPath):
    rawHandlers = require(handlersPath)  // gated by tool_policy
    result.handlers = prefixHandlers(parsed.name, rawHandlers)
  else:
    result.handlers = {}
  return result

function prefixToolSchema(skillName, schema):
  originalName = schema.function.name
  return {
    ...schema,
    function: {
      ...schema.function,
      name: `skill__${skillName}__${originalName}`
    }
  }

function prefixHandlers(skillName, handlers):
  result = {}
  for [key, fn] of Object.entries(handlers):
    result[`skill__${skillName}__${key}`] = fn
  return result

function enableSkill(root, skillName):
  overlay = loadOverlay(root)
  active = overlay.memory?.active_skills || []
  if not active.includes(skillName):
    active.push(skillName)
    overlay.memory.active_skills = active
    saveOverlay(root, overlay)

function disableSkill(root, skillName):
  overlay = loadOverlay(root)
  active = overlay.memory?.active_skills || []
  overlay.memory.active_skills = active.filter(s => s !== skillName)
  saveOverlay(root, overlay)

function buildSkillsPrompt(discovered, activeNames):
  lines = ['\nAvailable skills:']
  for skill of discovered:
    marker = activeNames.includes(skill.name) ? ' [ACTIVE]' : ''
    lines.push(`- ${skill.name}${marker}: ${skill.description}`)
  lines.push('')
  // inject active skill instructions
  for name of activeNames:
    skill = discovered.find(s => s.name === name)
    if not skill: continue
    loaded = loadSkill(skill.dir)
    lines.push(`<skill name="${name}">`)
    lines.push(loaded.instructions)
    lines.push(`</skill>`)
  return lines.join('\n')

function getSkillTools(root, activeNames):
  schemas = []
  handlers = {}
  discovered = discoverSkills(root)
  for name of activeNames:
    skill = discovered.find(s => s.name === name)
    if not skill: continue
    loaded = loadSkill(skill.dir)
    schemas.push(...loaded.tools)
    Object.assign(handlers, loaded.handlers)
  return { schemas, handlers }

--- src/agent.js modifications ---

  // existing commands...
  if command === '/skills': return handleSkillsCommand(root, arg, rl)

handleSkillsCommand(root, arg, rl):
  [sub, ...rest] = arg.split(/\s+/)
  if sub === 'list':

    activeNames = getActiveSkills(root)
```
