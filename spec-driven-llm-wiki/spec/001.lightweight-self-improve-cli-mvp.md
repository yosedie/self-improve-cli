# Spec 001: Lightweight Self-Improve CLI MVP

## Task/Prompt

User wants implementation start for an agentic AI coding tool with core differentiator: self-improving JSON runtime prompts/profiles. Must read and maintain `spec-driven-llm-wiki/AGENTS.md`. Tool must stay lightweight because comparable tools can use 500MB-1GB RAM; target is much lower memory and cross-platform.

## Tujuan

Buat MVP ringan yang menyediakan fondasi:

- JSON profile immutable base + mutable overlay.
- Growth policy: `none`, `low`, `medium`, `high`, `very_high`.
- Durable event log and patch audit.
- Deterministic self-improvement patch proposal/apply flow.
- Basic coding tools via CLI: read file, search files, run commands without shell dependency.
- No runtime dependencies, no Electron, no full repo indexing.

## Kenapa Tujuan Ini

Self-improvement perlu aman, reversible, dan murah RAM. MVP harus membuktikan profile engine dan growth gate dulu sebelum TUI/model/tool-calling lebih besar ditambah.

## Pseudocode

```text
init:
  create .selfimprove/
  copy profiles/default.profile.json to base.profile.json if absent
  create overlay.profile.json if absent
  create events.jsonl and patches.jsonl if absent

loadProfile:
  read base profile
  read overlay profile
  validate both
  deep merge base + overlay
  validate merged profile

observe/improve:
  append event
  build deterministic patch from failure/request text
  evaluate growth policy
  write patch audit
  if allowed auto-apply:
    apply JSON patch to overlay
    validate merged profile
    save overlay
  else:
    print patch for human review

tools:
  read: normalize path, read utf8, cap bytes
  search: recursive scan excluding heavy dirs, cap matches
  run: spawn executable with args, shell false
```
